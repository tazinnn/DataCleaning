# Data Cleaning
Data cleaned using SQL commands to make the data more usable.

Some simple tasks performed to practice some commonly used SQL functions.

### SQL Concepts Implemented
1. **CONVERT()**
2. **ISNULL()**
3. **SUBSTRING()**
4. **CHARINDEX()**
5. **LEN()**
6. **PARSENAME()**
7. **REPLACE()**
8. **CASE**
9. **CTE**
10. **ROW_NUMBER with PARTITION**
